Internet Explorer 6

Tout d'abord, ouvrir le menu Outils et cliquer sur "Options Internet..." (voir figure 1).

Figure 1: Menu Outils

La fenêtre "Options Internet" s'ouvre. Dans cette fenêtre (voir figure 2),

  • sélectionner l'onglet "Contenu",
  • cliquer sur le bouton ``Certificats.

Figure 2: Fenêtre "Options Internet" : Onglet "Contenu"

Une nouvelle fenêtre s'ouvre nommée "Certificats" (voir figure 3). Cette fenêtre va vous permettre d'ajouter le nouveau certificat. Cliquer sur le bouton "Importer" pour démarrer l'assistant d'importation de certificats (voir figure 4).

Figure 3: Fenêtre "Certificats"

Figure 4: Assistant d'importation 1/5"

Cliquer sur "Suivant". Cette page (voir figure 5) vous permet de charger le fichier correspondant au certificat. Cliquer sur Parcourir puis sélectionner le certificat. Attention l'extension est ".p12", pour la voir dans la liste des fichiers, sélectionner dans "Fichiers de type", le type "Echange d'informations personelles(*.pfx,*.p12)". Ouvrir le certificat.

Figure 5: Assistant d'importation 2/5"

Après ouverture de la clé, il est possible qu'un mot de passe soit demandé (voir figure 6). Si aucune indication concernant un mot de passe ne vous as été fourni, cliquer sur "Suivant" en laissant le champ texte vide.

Figure 6: Assistant d'importation 3/5"

L'étape suivante vous permet de choisir où placer le certficat. L'emplacement par défaut est satisfaisant, cliquer sur "Suivant".

Figure 7: Assistant d'importation 4/5"

Vous arrivez à la dernière étape de l'assistant d'importation de certificat (voir figure 8) : cliquer sur "Terminer".

Figure 8: Assistant d'importation 5/5"

Dans la fenêtre "Certificats", on constate la présence d'un certificat de plus dans la liste (voir figure 9). Cliquer sur "Fermer".

Figure 9: Fenêtre "Certificats"

L'installation du certficat est terminée. Vous pouvez vous rendre sur le site sécurisé. A la connection, il vous sera damnder de choisir un certificat (voir figure 10).

Figure 10: Choisir un certificat
